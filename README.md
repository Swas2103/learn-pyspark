# learn-pyspark
Hands-on PySpark learning repository with concepts, examples, and mini projects for big data processing.

This repository documents my learning journey with **PySpark**, focusing on big data processing, distributed computing concepts, and building scalable data pipelines.

The objective is to gain strong hands-on experience by practicing core concepts, experimenting with transformations, and implementing small but meaningful projects.

---

## 🎯 Goals

- Build a solid foundation in PySpark
- Understand distributed data processing concepts
- Apply Spark transformations and actions effectively
- Write clean, readable, and reusable PySpark code
- Progress toward building real-world ETL pipelines

---

## 🧠 Topics Covered

- PySpark fundamentals
- RDDs (Resilient Distributed Datasets)
- DataFrames & Spark SQL
- Transformations and actions
- Data cleaning and aggregation
- Partitioning and caching
- Performance optimization basics
- Introductory ETL workflows

---

## 🛠️ Tech Stack

- Python
- PySpark
- Apache Spark
- VS Code / Jupyter Notebook
